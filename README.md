# Lownheur

Lownheur est un SaaS qui permet à une personne d'organiser ses catégories, événements, objectifs et planifications depuis une interface web ou depuis une IA compatible MCP.

Le projet avance par versions validées. La V1 est actuellement en **spécification** : aucun développement fonctionnel ne commence avant validation de [`docs/versions/V1.md`](docs/versions/V1.md).

## Documents de référence

- [`docs/product/VISION.md`](docs/product/VISION.md) : vision long terme et limites des versions.
- [`docs/versions/V1.md`](docs/versions/V1.md) : contrat fonctionnel de la V1 à valider.
- [`docs/architecture/V1.md`](docs/architecture/V1.md) : architecture technique proposée.
- [`docs/TASKS_V1.md`](docs/TASKS_V1.md) : plan de réalisation et état réel.
- [`docs/WORKFLOW.md`](docs/WORKFLOW.md) : règles d'ingénierie et de validation.
- [`docs/DECISIONS.md`](docs/DECISIONS.md) : décisions produit et techniques.
- [`docs/CHANGELOG.md`](docs/CHANGELOG.md) : changements visibles par version.
- [`docs/JOURNAL.md`](docs/JOURNAL.md) : jalons du projet, sans journal verbeux.
